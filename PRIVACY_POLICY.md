Privacy Policy — Instagram Carousel Downloader
Last updated: March 2026
Data Collection
This extension does not collect, store, transmit, or share any personal data or browsing information.
How It Works
The extension operates entirely within the user's browser. When the user clicks "Fetch Media" on an Instagram post page, it makes a request to Instagram's API using the browser's existing session to retrieve media URLs. These URLs are then used to download files directly to the user's device.
No External Servers
This extension does not communicate with any third-party server or analytics service. All processing happens locally in the browser.
Permissions Used

activeTab: To detect if the current page is an Instagram post.
scripting: To execute a fetch request within the Instagram tab using the browser's existing authenticated session.
downloads: To save media files to the user's device.
host_permissions (instagram.com): To access the page context required to retrieve post media data.
